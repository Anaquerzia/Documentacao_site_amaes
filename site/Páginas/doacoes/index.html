    Página “Doações”

    Objetivo da página

    A página “Doações” orienta quem visita o site sobre como contribuir financeiramente com a AMAÊS Itabira-MG.
    Ela explica a importância da doação, apresenta os dados de PIX e oferece um botão para doações recorrentes (quando disponível).

    Nesta página, a administradora do site poderá atualizar textos, QR Code, dados bancários/PIX e o link do botão de doação.

    Como acessar a página “Doações” no WordPress

    1. No painel do WordPress, acesse o menu lateral esquerdo.
    2. Clique em Páginas.
    3. Procure pela página chamada Doações.
    4. Passe o mouse sobre o nome da página e clique em Editar para abrir o editor de blocos.

    Bloco 1 – Título e pergunta “Por que a sua doação é tão importante?”

    Na parte superior, a página exibe: - Título “Doações”. - Uma faixa colorida com a pergunta “Por que a sua doação é tão importante?”. - Um texto em destaque explicando o impacto das doações (por exemplo, “Porque juntos transformamos dor em ação…”).

    O que pode ser alterado

    • Título “Doações”.
    • Pergunta principal.
    • Texto da faixa de destaque explicando o impacto da doação.

    Como editar

    1. Clique no título “Doações” para ajustar o texto, se desejar.
    2. Clique na pergunta “Por que a sua doação é tão importante?” para editar a frase.
    3. Clique no texto da faixa ao lado (fundo preto) e ajuste a mensagem de impacto.
    4. Clique em Atualizar para salvar.

    Use frases curtas e fortes, que mostrem claramente por que a contribuição é importante.

    Bloco 2 – QR Code e dados do PIX

    A seção central mostra: - Um QR Code para doação via PIX. - O texto “Doe via PIX” com o CNPJ da associação. - Um texto explicando que é possível doar qualquer valor. - Um botão “Quero doar mensalmente” (quando houver essa opção).

    O que pode ser alterado

    • Imagem do QR Code.
    • Texto com os dados do PIX (chave, CNPJ).
    • Texto de orientação (por exemplo, “Use o QR code ao lado para doar via PIX qualquer valor uma única vez”).
    • Texto e link do botão de doação mensal.

    Como editar

    1. Clique na imagem do QR Code:
    2. Use Substituir para enviar um novo QR Code, caso o link de pagamento seja alterado.
    3. Verifique se o novo QR Code aponta para o destino correto.
    4. Clique no texto “Doe via PIX: …” e ajuste:
    5. Chave PIX (CNPJ, telefone, e-mail ou chave aleatória).
    6. Formatação, se necessário (mantendo claro para quem lê).
    7. Edite o texto explicativo logo abaixo, orientando a pessoa sobre como usar o QR Code ou os dados de PIX.
    8. Clique no botão “Quero doar mensalmente”:
    9. Ajuste o texto do botão, se necessário.
    10. Atualize o link para a página ou sistema de doação recorrente (ex.: formulário, plataforma externa).
    11. Clique em Atualizar para salvar.

    Sempre teste o QR Code com o celular e o botão de doação no site publicado, garantindo que ambos funcionam.

    Rodapé

    O rodapé com e-mail, Instagram, menu inferior e direitos autorais é compartilhado com as demais páginas.
    As instruções detalhadas para editar o rodapé estão na documentação da página Home e na seção de Manutenção geral.

    Checklist antes de salvar alterações em “Doações”

    Antes de clicar em Atualizar, a administradora do site deve conferir:

    • Se a chave PIX (CNPJ ou outra) está correta e atualizada.
    • Se o QR Code aponta para o destino certo (testar com o celular).
    • Se o texto explica claramente como doar (PIX único, doação mensal etc.).
    • Se o botão “Quero doar mensalmente” leva para o link correto.
    • Se não há erros de digitação nos dados financeiros (valores, CNPJ, e-mail).

    Depois de qualquer mudança nos dados de PIX ou links de doação, é recomendável informar a equipe responsável pelas finanças para que todos saibam que a página foi atualizada.
